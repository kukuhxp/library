QUESTION & ANSWER
-------------------------------------------------------------------
Q: Dimana parameter model disimpan?

A: Parameter model biasanya disimpan di server, seperti server OpenAI, Google Cloud, AWS, dll. Bentuk file model terkompresi seperti .bin, .pt, .ckpt, .onnx, .safetensors.

Contoh:

model/
 ├── config.json       ← pengaturan arsitektur model
 ├── pytorch_model.bin ← parameter model (ratusan GB untuk GPT)
 └── tokenizer.json     ← cara model memecah kata menjadi token
 
File-file tersebut berisi jutaan hingga triliunan angka float.

Contoh:

weight_1 = 0.1432
weight_2 = -0.9843
bias_1   = 0.0056

Angka-angka ini disusun menjadi matriks besar untuk tiap lapisan jaringan saraf. Itulah yang disebut parameter model, dan nilai-nilai itulah yang membuat AI ingat pola-pola dari hasil training.

Tahapan:

1. Server memuat parameter dari file model ke dalam RAM atau GPU memory.
2. Proses inference dilakukan, yaitu menghitung keluaran berdasarkan parameter tersebut.
3. Setelah itu, hasilnya dikirimkan ke pengguna.
