# PROGRAMMING LANGUAGE

Programming language atau bahasa pemrograman adalah bahasa khusus yang digunakan untuk memberi instruksi kepada komputer agar melakukan tugas tertentu, seperti menghitung, menampilkan data, menjalankan aplikasi, atau mengontrol perangkat keras.

## Low-level Language

Low-level language adalah bahasa pemrograman yang dekat dengan hardware atau instruksi mesin komputer. Bahasa ini memberikan kontrol penuh atas bagaimana komputer menjalankan operasi, tapi biasanya sulit dipahami dan kurang abstrak dibanding bahasa tingkat tinggi. Contohnya adalah Assembly dan Machine Code.

## Mid-level Language

Mid-level language adalah bahasa pemrograman yang berada di antara low-level dan high-level. Artinya, bahasa ini menawarkan abstraksi dari hardware untuk memudahkan programmer, tapi tetap memberikan kontrol cukup atas memori dan operasi komputer. Contohnya adalah C/C++ dan Rust.

## High-level Language

High-level language adalah bahasa pemrograman yang tinggi tingkat abstraksinya, jauh dari instruksi mesin, dan fokus pada kemudahan programmer untuk menulis logika aplikasi daripada mengurusi detail hardware. Contohnya adalah C#, Python, Java, dan JavaScript.
