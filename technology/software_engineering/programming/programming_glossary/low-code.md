# LOW-CODE

Definisi: Membuat aplikasi dengan sedikit kode, biasanya platform menyediakan antarmuka visual untuk bagian utama, tapi kita bisa menambahkan kode untuk menyesuaikan fitur atau logika kompleks.

Target pengguna: Developer atau tim IT yang ingin mempercepat pembuatan aplikasi tanpa menulis kode dari awal.

Contoh platform:

OutSystems, Mendix, Microsoft Power Apps


Kelebihan: Lebih fleksibel dibanding no-code, tetap cepat.

Kekurangan: Masih memerlukan pengetahuan teknis untuk fitur tertentu.
