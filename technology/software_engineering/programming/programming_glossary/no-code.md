# NO-CODE

No-code adalah proses membuat aplikasi tanpa menulis kode sama sekali. Semua dilakukan melalui drag-and-drop, konfigurasi visual, dan logika siap pakai. Target penggunanya adalah orang non-teknis atau bisnis yang ingin membuat aplikasi tanpa belajar pemrograman.

Example:

Wix, Airtable, dan Bubble
