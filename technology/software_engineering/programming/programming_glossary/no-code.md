# NO-CODE

Definisi: Membuat aplikasi tanpa menulis kode sama sekali. Semua dilakukan melalui drag-and-drop, konfigurasi visual, dan logika siap pakai.

Target pengguna: Orang non-teknis atau bisnis yang ingin membuat aplikasi tanpa belajar pemrograman.

Contoh platform:

Wix / Webflow: Untuk membuat website.

Airtable: Untuk database atau aplikasi ringan.

Bubble: Untuk aplikasi web interaktif.


Kelebihan: Cepat, mudah, tidak perlu programmer.

Kekurangan: Kurang fleksibel, tergantung pada fitur yang disediakan platform.
