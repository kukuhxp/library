# DATE

Date adalah objek bawaan di JavaScript yang digunakan untuk mewakili tanggal dan waktu.

## Fungsi Date

1. Menampilkan waktu sekarang.
2. Mengambil tahun, bulan, hari, jam, menit, detik.
3. Menghitung selisih waktu.
4. Mengatur tanggal tertentu.

## Date Global Function

Example:

```
console.log(Date());

// Output
"Wed Nov 12 2025 07:45:30 GMT+0700 (Western Indonesia Time)"

```

## Date Constructor Function

Example:

```
const d = new Date();

d.property;
d.method();
```
