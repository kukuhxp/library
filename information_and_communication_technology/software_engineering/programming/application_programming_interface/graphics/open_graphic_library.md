# OPEN GRAPHICS LIBRARY (OPENGL)

Open Graphics Library (OpenGL) adalah API grafis yang digunakan untuk membuat aplikasi 2D dan 3D seperti game, software desain, simulasi, dan visualisasi. OpenGL menyediakan perintah-perintah standar agar aplikasi bisa menggambar bentuk, tekstur, cahaya, dan objek 3D di layar.

Dikembangkan oleh Khronos Group, versi awal dibuat oleh: Silicon Graphics (SGI). OpenGL adalah API grafis cross-platform yang sudah ada sejak 1990-an dan masih banyak digunakan.

## Function

1. Render 2D/3D real-time.

2. Shader processing.

3. Membuat game.

4. Membuat antarmuka 3D.

5. Visualisasi ilmiah.

6. Aplikasi CAD dan grafik teknik.


## Example

1. Blender.

2. Google Earth.

3. Unity.
