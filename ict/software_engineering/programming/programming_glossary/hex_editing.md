# HEX EDITING

Hex editing adalah proses mengedit file dalam bentuk heksadesimal, yaitu representasi angka basis-16 dari data biner. Artinya, kamu membuka file—apa pun formatnya seperti e.xe, .dll, .jpg, .mp3, save game, firmware, .apk—dan melihat isi file tersebut sebagai angka-angka hex

Example:

`4D 5A 90 00 03 00 00 00 ...`
