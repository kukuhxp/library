# ANTI-TAMPER

Anti-tamper adalah teknik atau sistem yang dibuat untuk mencegah, mendeteksi, atau mempersulit upaya modifikasi tidak sah pada suatu perangkat, software, file, atau sistem. Tujuannya adalah melindungi integritas dan mencegah orang lain mengubah, membajak, memodifikasi, atau membongkar sesuatu tanpa izin.

Example:

1. Denuvo Anti-tamper
2. DotNet Reactor Anti-tamper 
3. Themida, VMProtect, Code Virtualizer

## Anti-tamper Methods

1. Obfuscation
2. Integrity check
3. Debugger detection
4. Anti-dumping


## Anti-tamper Hardware

1. Segel atau stiker yang rusak jika dibuka
2. Mekanisme yang menghapus data jika casing dibongkar
3. Chip fisik dengan sensor cahaya, tekanan, atau panas
