# LOW-CODE

Low-code adalah proses membuat aplikasi dengan sedikit kode, biasanya platform menyediakan antarmuka visual untuk bagian utama, tapi kita bisa menambahkan kode untuk menyesuaikan fitur atau logika kompleks. Target penggunanya adalah developer atau tim IT yang ingin mempercepat pembuatan aplikasi tanpa menulis kode dari awal.

Example:

OutSystems, Mendix, Microsoft Power Apps
