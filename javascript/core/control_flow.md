# CONTROL FLOW

## If...Else

**One Condition:**

```
if(condition) {
  statmeents;
}
```

**Two Condition:**

```
if(condition) {
  statmeents;
}
else {
  statmeents;
}
```

**Three or More Condition:**

```
if(condition 1) {
  statmeents;
} else if(condition 2) {
  statmeents;
} else {
  statements;
}
```

## Switch...Break

```
switch(object) {
 case value:
   statements;
   break;
 default:
   statements;
}
```
