# MONTH CODE

Month code adalah huruf satu karakter yang digunakan untuk mewakili expiry month dari futures contract. Berikut ini adalah kode-kode pada bulan futures contracts:

- F - Januari

- G - Februari

- H - Maret

- J - April

- K - Mei

- M - Juni

- N - Juli

- Q - Agustus

- U - September

- V - Oktober

- X - November

- Z - Desember
