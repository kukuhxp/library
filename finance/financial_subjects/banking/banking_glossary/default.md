# DEFAULT

Default adalah kondisi ketika pihak yang berutang gagal memenuhi kewajiban keuangannya sesuai perjanjian.

## 1. Payment Default

Gagal membayar kewajiban finansial tepat waktu.   
Example: tidak membayar cicilan kredit atau bunga obligasi.

## 2. Technical Default

Pembayaran masih dilakukan, tetapi melanggar klausul kontrak.   
Example: rasio utang melebihi batas yang disepakati.

## 3. Sovereign Default

Default yang dilakukan oleh negara.   
Example: negara gagal membayar utang luar negeri.
