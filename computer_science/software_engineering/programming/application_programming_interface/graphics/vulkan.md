# VULKAN

Vulkan adalah API grafis dan komputasi tingkat rendah (low-level API) yang digunakan untuk membuat aplikasi dengan grafis berkinerja tinggi, seperti game, engine 3D, simulasi, VR/AR, dan aplikasi yang membutuhkan akses langsung ke GPU.

Dikembangkan oleh Khronos Group, organisasi yang sama yang mengelola OpenGL, WebGL, OpenCL. Vulkan adalah API cross-platform yang memberi developer kontrol sangat dekat ke hardware GPU sehingga performanya bisa lebih optimal dibandingkan API tradisional seperti OpenGL atau DirectX 11.

## Function

1. Mengelola rendering 2D/3D.

2. Mengontrol memori GPU.

3. Mengatur shader, pipeline grafis, komputasi GPU.

4. Memungkinkan multi-threading lebih efisien.

5. Performa tinggi dengan overhead CPU rendah.
